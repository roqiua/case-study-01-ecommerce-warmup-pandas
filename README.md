# Case Study 01 - E-Commerce Warmup (Pandas)

This repository contains warm-up exercises using **Python** and **Pandas** on an **E-Commerce Purchases** dataset. The goal is to practice essential data analysis techniques and strengthen Pandas skills through real-world questions.

## 📌 Objectives

- Load and inspect datasets
- Explore data using Pandas
- Filter and query data
- Perform aggregations and grouping
- Work with string operations
- Extract insights from purchase data

## 🛠️ Technologies Used

- Python
- Pandas
- Jupyter Notebook

- 
## 📂 Dataset
The dataset used in this repository is the **E-Commerce Purchases** dataset available on Kaggle.

**Dataset Link:**  
[:contentReference[oaicite:0]{index=0}](https://www.kaggle.com/datasets/utkarsharya/ecommerce-purchases/code)

The dataset includes customer purchase information such as:
- Purchase Price
- Credit Card Provider
- Email
- Job
- Language
- Browser Information
- IP Address
- Company
- And more...


## 📖 Topics Covered

- Data Inspection
- Filtering Rows
- Selecting Columns
- Boolean Indexing
- Sorting
- GroupBy Operations
- Value Counts
- Unique Values
- String Methods
- Basic Statistical Analysis

## 🎯 Purpose

This case study is part of my **Data Analysis with Python** practice journey. It focuses on improving problem-solving skills and gaining hands-on experience with Pandas before working on larger data analysis projects.
